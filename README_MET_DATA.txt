The {location}.csv files are an array of 11 rows by 32 columns.
The timesteps are 3 hourly intervals (0,3,6,12,15,18,21) from the last forecast. 

The variables are (in order:)

Precipitation Probabilty
Wind Direction
Wind Gust
Feels Like Temperature
Relative Humidity
Wind Speed
Max UV Index
Temperature
Weather Type
Visibility
Minutes in the Day
